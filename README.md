# sunreef-tasks

clone or download repository
``` bash
git clone https://github.com/dodocanfly/sunreef-tasks.git
```

go to script folder
``` bash
cd sunreef-tasks
```

run composer install
``` bash
composer install
```

run example script
``` bash
php examples/task1.php
php examples/task2.php
php examples/task3.php
php examples/task4.php
```
